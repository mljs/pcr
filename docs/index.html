Creates new PCR (Principal component regression)

new PCR(predictor: Array, response: Array, options: any, pcaWeight: number, intercept: boolean)
Parameters
predictor (Array) matrix with predictor variables. (Each column is an array)
response (Array) matrix with response variables. (Each column is an array)
options (any = {})
pcaWeight (number) Weight to choose the principal components. It refers to the weight that components must sum with each other (in percent) to perform the regression.
intercept (boolean) Intercept
